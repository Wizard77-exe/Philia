# Philia

> *"She started as a simple chatbot. She became my way of learning Computer Science."*

## About

Philia is a chatbot written entirely in **C**.

This project did not begin with the goal of building the most advanced AI.
It began because I wanted to learn C deeply.

Instead of writing dozens of unrelated mini-projects, I decided to dedicate everything I learn into one long-term project. Every new concept I study becomes another piece of Philia.

As Philia grows, so do I.

---

## Purpose

This repository is my personal journey of learning:

- C Programming
- Data Structures
- Memory Management
- Natural Language Processing (NLP)
- Machine Learning
- Information Retrieval
- Algorithms
- Mathematics behind AI

Every version represents something new that I have learned.

---

## Current Philosophy

I don't want to simply *use* algorithms.

I want to understand them.

If Philia uses TF-IDF, I want to know why TF-IDF exists.

If Philia uses Cosine Similarity, I want to understand vectors and the mathematics behind it.

If Philia uses Transformers someday, I want to understand every step that led there.

This project is built from first principles whenever possible.

## Long-Term Vision

Philia is more than a chatbot project.

The long-term goal is to build an AI capable of understanding not only language but eventually environments, objects, memories, and interactions.

This journey will likely expand into:

- Machine Learning

- Simulation

## Why C?

People often build AI projects using Python.

I chose C because I wanted to understand what happens underneath.

Memory.
Pointers.
Performance.
Data structures.

Building Philia in C forces me to understand the fundamentals before relying on higher-level libraries.

---

## Development Style

I don't rush features.

Every version exists because I learned something.

Sometimes a whole version only introduces one mathematical concept.

Sometimes I spend days reorganizing the codebase.

Sometimes I spend weeks studying before writing a single line.

That's intentional.
